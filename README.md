# Introducing... Andromedaaa!

Andromedaaa is a website that retrieves information from the NASA API; keeps track of the title, explanation, image/video source, and copyright in a very presentable format; and allows for the user to find information from the NASA API from the past using our new feature: the date picker! It is named after the Scouting/Strategy Database that I created for FRC Team 6135 Arctos last season!
