# Introducing... Andromedaaa!

Andromedaaa is a website that retrieves information from the NASA API; keeps track of the title, explanation, image/video source, and copyright in a very presentable format; and allows for the user to find information from the NASA API from the past using our new feature: the date picker! It is named after the Scouting/Strategy Database that I created for FRC Team 6135 Arctos last season!

## Features
- Data from the NASA API retrieved onto this website
- **Title, Explanation, Image/Video/YouTube Video/Copyright Credits** from each and every single day within the given dates on the NASA API styled with CSS and displayed on the website.
- A background image of the James Webb Telescope styled using the CSS property `background-image` and various other properties that allowed the background to appear as shown.
- A date changer, with a lot of additional styling, to allow the website to retrieve information from the NASA API at a specific date.
